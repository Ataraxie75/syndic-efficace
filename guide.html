Modifier mon site maintenant

Comment ça marche ?

1

Ouvrez votre espace d'administration

Allez sur /admin de votre site — ou cliquez directement sur le bouton ci-dessus.
Pensez à mettre cette page en favori dans votre navigateur !

Conseil : Bookmarkez cette page guide aussi — vous pourrez toujours y revenir si vous avez un doute.
2

Connectez-vous avec votre email et mot de passe

Entrez votre email et le mot de passe que vous avez choisi lors de votre première connexion.
Votre navigateur s'en souvient — vous n'aurez à le faire qu'une seule fois.

Mot de passe oublié ? Cliquez sur "Mot de passe oublié" sur la page de connexion. Vous recevrez un email de réinitialisation.
3

Choisissez la section à modifier

Cliquez sur "Informations du cabinet", puis déroulez la section que vous souhaitez mettre à jour :

— Avocate : votre nom, biographie, diplômes, photo
— Coordonnées : téléphone, email, adresse, horaires
— Page Copropriété : tous les textes de cette page
— Page Transition énergétique : tous les textes de cette page
— Informations légales : SIRET, adresse légale

Tout est en français, tout est expliqué
4

Modifiez, puis cliquez sur "Publier"

Cliquez dans le champ que vous souhaitez modifier, changez le texte, puis cliquez sur le bouton bleu "Publier" en haut à droite.

C'est tout. Votre site se met à jour automatiquement en 30 secondes.

Pas de panique : si vous faites une erreur, vous pouvez corriger et republier autant de fois que vous voulez.
